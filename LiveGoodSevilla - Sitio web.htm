El editor de Canva tiene varias secciones: la navegación principal, el panel lateral, la barra de herramientas, el lienzo y los controles de página. Para enfocarte en una sección y conocer más sobre cada una, utiliza el tabulador. Para ir al lienzo, presiona “Ctrl+F2”.
La navegación principal contiene botones con opciones para trabajar sobre el documento. Para pasar de un botón a otro, utiliza las teclas de flechas a la izquierda y a la derecha y, para activar un botón, utiliza la tecla Intro. Para ir al lienzo, presiona “Ctrl+F2”.
Lienzo
El lienzo es donde está tu diseño. Para ir a la página, presiona la tecla Intro. Para ir a grupos o elementos en la página, vuelve a presionar la tecla Intro. Presiona el tabulador para ir a la siguiente página, al siguiente grupo o al siguiente elemento. Para utilizar la barra de herramientas, usa la tecla “Ctrl+F1”. Si hay un elemento seleccionado, lo puedes mover con las teclas de flechas; lo puedes rotar con la teclas “,” y “.”; y puedes cambiar su tamaño con las teclas de flechas y la tecla “Ctrl”. Aumenta los incrementos de todos estos atajos con la tecla “Mayús.”.
